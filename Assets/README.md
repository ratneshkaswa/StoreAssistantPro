# Assets Folder

Drop your logo files here and **rebuild** — the app picks them up automatically.

| File | Purpose | Suggested Size |
|------|---------|---------------|
| `logo-64.png` | Login/Setup hero, sidebar | 64×64 px |
| `logo-128.png` | Dashboard, about dialog | 128×128 px |
| `logo-256.png` | Splash/installer | 256×256 px |
| `app.ico` | Window icon, taskbar | Multi-res (16/32/48/256) |

All files should have transparent backgrounds (PNG-32).

**Important:** After placing the files, do a full Rebuild (Ctrl+Shift+B).
If no `logo-64.png` is found, a 🏪 emoji fallback is displayed.
